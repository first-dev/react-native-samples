# react-native-samples

This repository contains a collection of samples from my previous React Native projects.

The repository itself doesn't produce any runnable code, but it can be used as a reference.

## Index

- [app](app): Screen samples using expo-router
- [atoms](atoms): Recoil atoms
- [components](components): Generic React Native components
- [hooks](hooks): Custom hooks
- [i18n](i18n): Internationalization samples
- [plugins](plugins): Expo config plugins
- [queries](queries): react-query custom queries
- [redux](redux): Redux and RTK Query samples
- [screens](screens): Screen samples using react-navigation
- [types](types): TypeScript type definitions
- [utils](utils): Generic utility functions
